# My Personal AI PA Bot 🤖

A personal assistant Telegram bot powered by Groq (Llama 3 AI).

## Features
- 🧠 Ask anything — learning, questions, advice
- 📋 Save notes by saying "remember..." or "note..."
- 💬 Remembers conversation history
- 🌐 Works in English and Malay

## Commands
- /start — welcome message
- /notes — see all saved notes
- /clear — clear chat history
- /help — show help

## Built with
- Python
- python-telegram-bot
- Groq API (Llama 3)
- Railway (hosting)
